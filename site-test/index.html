Between lectures, labs, group projects, long walks across campus, and doing everything on a schedule that changes weekly, it’s easy for one missing item to turn into wasted time and extra stress.

Why this exists

UCSD isn’t difficult only because of one class. It’s difficult because everything stacks at once. Deadlines overlap, you lose time commuting across campus, and the day keeps going even when you’re tired.

This project is built around one idea: remove avoidable friction. If you can solve minor issues immediately, you protect time and attention for the work that actually matters.

What we consider “essential”

Stability: items that prevent small disruptions from becoming hour-long problems.

Continuity: basics like hydration, nutrition, and device power.

Focus: supplies that protect attention and reduce unnecessary distractions.

How we source items

Procurement is intentionally simple. If something is truly essential, it must be easy to replace quickly. In practice, that means we walk to Target.

The point is reliability: no complicated suppliers, no special orders, and no waiting weeks. If you need something that isn’t listed, submit a request and we’ll review it for practical value and availability.
